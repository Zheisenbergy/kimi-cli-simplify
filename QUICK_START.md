# Kimi CLI Simplify - 新手快速入门

## 什么是这个程序？

这是一个 AI 助手，可以：
- 和你对话（像 ChatGPT）
- 读取和修改你的文件
- 执行命令
- 搜索网页
- 记住你们的对话历史

## 安装和运行

### 1. 配置 API Key

创建 `.env` 文件：
```bash
cd /Users/zheisenbergy/code/agent/kimi-cli-simplify
echo 'OPENAI_API_KEY=sk-你的API密钥' > .env
```

### 2. 安装依赖

```bash
# 使用 pip
pip install -e .

# 或使用 uv
uv sync
```

### 3. 运行

```bash
python -m kimi_simplify.main
```

## 上下文压缩功能

### 什么是上下文压缩？

对话久了，历史消息会越来越多，导致：
- 超过 AI 的处理上限
- 响应变慢
- 费用增加

**压缩就是：把早期的长对话变成简短的摘要**

### 两种压缩方式

#### 1. 简单压缩（/compact）
**原理**：按规则提取关键信息
- 提取用户的问题
- 记录使用了什么工具
- 丢弃具体的工具输出

**优点**：免费、快速
**缺点**：质量一般

**使用时机**：消息较多但不太重要时

```
👤 You: /compact
✅ 简单压缩完成:
   消息: 15 → 6 条
   Token: ~5000 → ~1500
```

#### 2. 智能压缩（/compact-ai）
**原理**：用 AI 理解内容，生成高质量摘要
- 识别当前在做什么任务
- 保留错误和解决方案
- 记录代码的最终状态

**优点**：质量高、信息完整
**缺点**：需要调用 API（有成本）

**使用时机**：对话很重要，需要保留关键信息

```
👤 You: /compact-ai
🤖 正在使用 AI 生成智能摘要...
   （这会消耗一些 Token）

  [压缩] 正在生成智能摘要... 完成
✅ 智能压缩完成:
   消息: 20 → 4 条
   Token: ~8000 → ~1200
   节省: ~6800 Token
```

### 什么时候需要压缩？

程序会自动提示：
```
⚠️  提示：上下文较长（~9000 Token）
    建议使用 /compact 或 /compact-ai 压缩
```

或者你可以主动查看：
```
👤 You: /stats
📊 上下文统计:
  总消息数: 25
    - 系统提示: 1
    - 用户消息: 10
    - AI 回复: 9
    - 工具结果: 5
  估算 Token: ~9500

⚠️  Token 较多，建议压缩
```

## 流式输出

### 什么是流式输出？

**普通输出**：等待 AI 写完所有内容，一次性显示
**流式输出**：AI 写一个字，立即显示一个字

### 效果对比

普通模式（等待 5 秒）：
```
👤 You: 写首诗
（等待 5 秒）
🤖 Kimi: [完整的诗一次性出现]
```

流式模式（立即响应）：
```
👤 You: 写首诗
🤖 Kimi: 春 [0.1秒]
春 风 [0.2秒]
春 风 拂 [0.3秒]
春 风 拂 柳 [0.4秒]
... 逐字出现
```

### 好处

1. **感知更快**：不用等待很久
2. **体验自然**：像真人打字
3. **实时反馈**：知道 AI 在思考

## 会话持久化

### 什么是会话持久化？

程序会记住你们的对话，即使关闭后重新打开。

### 自动保存

对话自动保存在 `.kimi_session.json` 文件中。

### 手动控制

```bash
# 保存会话
👤 You: /save
✅ 会话已保存: /path/to/.kimi_session.json

# 加载会话
👤 You: /load
✅ 会话已加载，历史消息: 15 条
```

### 下次启动

```bash
$ python -m kimi_simplify.main
💾 发现已有会话: /path/to/.kimi_session.json
✅ 已加载会话，历史消息: 15 条

👤 You: 刚才我说了什么？  <- Kimi 记得！
```

## 常见问题

### Q: 压缩会丢失信息吗？
A: 简单压缩可能丢失细节，智能压缩会尽量保留关键信息。如果对话很重要，建议先 `/save` 保存，再压缩。

### Q: 智能压缩要花多少钱？
A: 取决于对话长度。通常 8000 Token 的压缩成本约 $0.02-0.05（人民币 0.15-0.35 元）。

### Q: 什么时候会自动压缩？
A: 目前需要手动触发。当 Token 超过 8000 时会提示建议压缩。

### Q: 压缩后可以恢复吗？
A: 不可以。压缩是单向的。重要对话请先 `/save` 备份。

### Q: 流式输出可以关闭吗？
A: 当前版本不支持关闭，这是默认行为。
